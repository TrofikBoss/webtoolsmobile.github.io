Плагин для упрощения мобильной вёрстки WebToolsMobile от TitanExpert

WebToolsMobile - это JavaScript плагин для сайта, который упрощает разработку мобильной версии для вашего сайта. Благодоря плагину можно будет делать мобильные версии сайтов намного быстрее, а новичкам намного легче. В плагине есть несколько встроенных команд, с помощью которых можно за минуту сделать открывающееся меню без каких либо знаний JavaScript и многое другое. Рекомендуется смотреть страницу с ПК

  • Первый элемент
  • Второй элемент
  • Третий элемент
  • Четвёртый элемент
  • Пятый элемент
Чтобы создать меню, нужно написать HTML разметку меню, CSS стили и самое главное JavaScript код, с помощью которого будет происходить открытие и закрытие меню по нажатию кнопки. Это занимает время, особенно у начинающих веб разработчиков. Благодоря нашему плагину создать такое меню можно написав всего лишь пару строчек кода

Установка плагина WebToolsMobile

1. Создайте пустой тег скрипта в HTML с типом "module"

<script type="module">


</script>

2. Подключите плагин и импортируйте его функции

1) Подключить плагин по веб ссылке

<script type="module">

import * as WTM from "https://trofikboss.github.io/webtoolsmobile.github.io/webtoolsmobile.js"

</script>

2) Скачать плагин и подключить его через локальные файлы

<script type="module">

import * as WTM from "./webtoolsmobile.js"

</script>

Работа с плагином

Для работы с нашим плагином в скрипт нужно написать нужные вам команды. В нашем плагине есть самые необходимые команды для создания мобильной версии сайта.

Создание меню - WTM.menuAdd()

    Пример меню
  • Первый элемент
  • Второй элемент
  • Третий элемент
  • Четвёртый элемент
  • Пятый элемент
На большинстве сайтов есть меню, и на мобильных версиях они особенные. Меню на мобильных версиях должны открываться и закрываться по нажатию на кнопку. Для создания меню есть команда WTM.menuAdd()
Эта функция при помощи JavaScript объединяет кнопку и меню, после чего кнопка может открывать и закрывать меню

<script type="module">

import * as WTM from "./webtoolsmobile.js"

WTM.menuAdd({"idButton": "button-1", "idMenu": "menu-1", "hiddenStyle": "display: none"})

</script>

У функции есть несколько атрибутов:
idButton - id кнопки*
idMenu - id меню*
hiddenStyle - css стиль, который накладывается на меню при его исчезновении. Автоматически задано значение "display: none", можно задать любой стиль, который будет убирать меню(opacity: 0)

Все меню на этом сайте реализованы через плагин WebToolsMobile
(стили меню и крутые анимации не входят в плагин)

Скрытие элементов - WTM.hideElement()

На мобильных версиях сайта могут быть такие элементы, которые никак не вписываются под мобильные устройства. Может быть и наоборот, могут быть элементы, которые подходят только для мобильной версии. Эта функция поможет скрывать нужные вам элементы.

<script type="module">

import * as WTM from "./webtoolsmobile.js"

WTM.hideElement({"selector": ".menu__button", "hideWidth": 600, "mod": "bigger", "test": false})

</script>

У функции есть несколько атрибутов:
selector - css селектор, по которому можно найти элемент в HTML. (.menu #menu-1)*
hideWidth - ширина экрана, при которой будет скрываться элемент*
mod - режим работы функции bigger/smaller
- bigger - скрываются элементы, когда экран больше указанного значения
- smaller - скрываются элементы, когда экран меньше (стандартное значение)
test - тест функции при любых значениях (если true, автоматически false)

Не все параметры в функциях нашего плагина являются обязательными. У необязательных параметров значение установлено автоматически, и вы можете их написать, если вам нужно их поменять.

Скролинг сайта - WTM.scrollButton()

На мобильных версиях сайта иногда бывают кнопки, которые могут скролить сайт наверх, когда это необходимо. Мы так же предусмотрели эту функцию

<script type="module">

import * as WTM from "./webtoolsmobile.js"

WTM.scrollButton({"selector": ".scroll__button", "scroll": 0, "showOn": 500, "position": "absolute", "smoothScroll": true, "setStyle": 'position: fixed; right: 20px; bottom: 20px; z-index: 2'})

</script>

У функции есть несколько атрибутов:
selector - селектор, по которому можно найти элемент*
scroll - позиция прокрутки, до которой прокручивается сайт*
showOn - позиция прокрутки, на которой будет появляться кнопка
position - позиция, относительно которой будет совершена прокрутка absolute/relative
- absolute - прокрутка делается относительно сайта на точную координату (стандартное значение)
- relative - прокрутка делается относительно от текущей прокрутки на определённую позицию
smoothScroll - плавная прокрутка (автоматически true)
setStyle - дополнительные стили для кнопки, назначаются автоматически, можно изменить вручную

Перестройка элементов - WTM.transform()

Для мобильных версий сайта очень часто нужно менять параметры элементов, чтобы они подходили под текущий размер экрана. Эта функция поможет вам!!

<script type="module">

import * as WTM from "./webtoolsmobile.js"

WTM.transform({"transformWidth": 700, "mod": "smaller", "setStyle": "#test-transform {border: 3px solid orange; background-color: white}", "test": "false"})

</script>

У функции есть несколько атрибутов:
transformWidth - ширина экрана, при которой происходит изменение элементов*
mod - режим, по которому происходит трансформация bigger/smaller
- bigger - изменения происходят с объектами больше указанной величины
- smaller - изменения происходят с объектами меньше указанной величины (автоматически)
setStyle - css стили, которые будут назначены элементам*
test - применяет свойства к объектам и тестирует их без учёта условий функции (автоматически false)

Атрибуты функции можно расставлять в любом порядке
